概述
----

AhaRobot 机器人遥操作系统是一个完整的机器人控制解决方案，包含以下核心组件：

- **视觉跟踪模块** (VisionTracker): 实时目标检测和跟踪
- **逆运动学求解器** (IKSolver): 将笛卡尔空间位置转换为关节角度
- **组织映射模块** (OrgMap): 坐标系转换和映射
- **主控制程序** (main_teleop.py): 集成所有模块的主程序

工作流程
--------

1. **视觉感知**: 摄像头捕获图像，视觉跟踪模块检测目标
2. **坐标映射**: 将图像坐标映射到机器人工作空间
3. **逆运动学**: 计算达到目标位置所需的关节角度
4. **机器人控制**: 通过 MuJoCo 仿真器控制机器人运动
5. **反馈循环**: 实时更新目标位置，形成闭环控制

主要特性
--------

🤖 **高精度仿真**
   基于 MuJoCo 物理引擎，提供逼真的机器人仿真环境

👁️ **实时视觉跟踪**
   集成 OpenCV，支持多种目标检测和跟踪算法

🔧 **智能逆运动学**
   高效的数值求解算法，支持复杂机器人结构

🎮 **直观控制接口**
   简洁的 API 设计，易于集成和扩展

📊 **实时监控**
   提供详细的系统状态和性能指标

使用场景
--------

- **机器人遥操作**: 通过视觉反馈进行远程机器人控制
- **自动化装配**: 基于视觉的精确装配任务
- **机器人学习**: 为机器人学习算法提供仿真环境
- **研究开发**: 机器人控制算法的快速原型开发
